# PDF Text Writer

PDFの上に文字を追加し、透かしやアプリ名のスタンプを付けずに保存するブラウザアプリです。PDFファイルはサーバーへ送信せず、読み込み・編集・保存をすべてブラウザ内で処理します。

## 主な機能

- 複数ページPDFの表示
- テキストの追加、編集、押したままドラッグ移動、削除
- 文字サイズ、文字色、太字、文字揃え
- PC・スマートフォン共通のカラーパレット
- ズーム表示
- 全ページの90度回転（保存PDFにも反映）
- 日本語フォントの埋め込み
- PC、スマートフォン、タブレット対応
- 透かしなしPDFの保存

## 起動方法

ES ModulesとPDF.jsのWorkerを使用するため、ファイルを直接開かずローカルHTTPサーバーから起動してください。

```powershell
cd pdf-text-writer
node dev-server.js
```

ブラウザで `http://127.0.0.1:4173/` を開きます。

## 使い方

1. 「PDFを開く」で端末内のPDFを選択します。
2. 「テキストを追加」を押し、PDF上の追加位置をクリックします。
3. テキストを入力し、必要に応じてスタイルや位置を調整します。
4. 「PDFを保存」を押すと、元ファイル名に `-text` を付けたPDFを保存します。

## 外部ライブラリ

- [PDF.js](https://mozilla.github.io/pdf.js/) — PDF表示
- [pdf-lib](https://pdf-lib.js.org/) — PDF書き出し
- [@pdf-lib/fontkit](https://github.com/Hopding/fontkit) — カスタムフォント埋め込み
- [Noto Sans JP](https://fontsource.org/fonts/noto-sans-jp) — 日本語フォント

初期版はライブラリと日本語フォントをCDNから読み込みます。そのため、初回利用時にはインターネット接続が必要です。PDF本体が外部へ送信されることはありません。

## 仕様上の注意

- 既存のPDF文字は直接編集しません。新しい文字を上から追記します。
- 保存した文字はPDFへ焼き込まれるため、再読み込み後はテキストボックスとして再編集できません。
- パスワード保護PDFには対応していません。
